fogLoader

Ajax loader for jQuery utilizing themes

Download .zip Download .tar.gz View on GitHub

fogLoader is an easy to use modal ajax loader that works with your existing jQuery theme

You will need a reference to jQuery 1.10 and jQueryUI 1.10.3, plus your theme.

Simply add a single div and you are all set <div id="myLoader"></div>

Demo


$('#myLoader').fogLoader();
